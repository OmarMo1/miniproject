Hoe start je de applicatie
--------------------------

1. Open de Python file die zich in dit .ZIP bestand bevind. ( zorg ervoor datt alle bestanden zich in dezelfde map bevinden om fouten te voorkomen. )

2. U dient ervoor te zorgen dat IDE beschikt over de volgende modules: XMLtoDict, Requests.

3. Eenmaal geopend gebruik "SHIFT + F10" om de applicatie te runnen.
   (alternatieve optie om de applicatie te laten runnen is door op het
   groene pijltje rechtsboven in het scherm te klikken).

4. De applicatie heeft zich nu in een venster geopend.
